1. npm install
   - завантаження розширень/бібліотек
2. node create-user.js
   - треба змінити хост, юзер і пароль на свої, ВАЖЛИВО цей юзер повинен мати права на підключення з будь якого хосту
3. npm run init-db
   - в дб-конфіг треба змінити хост, імя і пароль(на ті які указала в CREATE USER команді) 
4. npm start
   - запуск серверу
5. ДЛЯ ТЕСТУВАННЯ без ФЕ:
	curl -X POST http://localhost:3000/add-item -H "Content-Type: application/json" -d '{"name": "Milk"}'
   додасть айтем в таблицю, запустити в іншому вікні, в першому вікні повинен біжати сервер
